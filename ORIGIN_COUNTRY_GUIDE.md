# 原産国の判定ルール

## 🌍 トレーディングカードの原産国

### 基本原則
**原産国 = 製造国（印刷された国）**

カードに書かれている言語やブランドの国籍ではなく、**実際に印刷・製造された国**が原産国です。

---

## 🔍 製造国の確認方法

### 1. カード裏面をチェック
多くのカードには裏面や側面に製造国が印刷されています:
```
例: "Made in Japan"
例: "Printed in Belgium"
例: "© 2023 Pokémon. Manufactured by The Pokémon Company International, Inc."
```

### 2. パッケージをチェック
未開封品の場合、パッケージに製造国が記載されています:
```
例: "Made in USA"
例: "Product of China"
```

### 3. 版元の情報
- **日本製**: 「株式会社ポケモン」「コナミデジタルエンタテインメント」
- **米国製**: "The Pokémon Company International"
- **中国製**: OEM製造（パッケージに記載）

---

## ⚠️ よくある間違い

### ❌ 間違い1: 言語 = 原産国
```
英語のポケモンカード → 必ずしも米国製ではない
→ ベルギー製、中国製の可能性もある
```

### ❌ 間違い2: ブランドの国 = 原産国
```
ポケモン（日本企業） → 必ずしも日本製ではない
→ 米国版は米国製、中国製もある
```

### ❌ 間違い3: 販売地域 = 原産国
```
米国で販売 → 必ずしも米国製ではない
→ 中国やベルギーで製造されることも
```

---

## ✅ 正しい判定フロー

```
1. カード裏面/側面を確認
   ↓
2. "Made in XXX" の記載を探す
   ↓
3. パッケージを確認
   ↓
4. 不明な場合は版元の公式情報を確認
   ↓
5. それでも不明なら「CN」（中国）とする
   （OEM製造が多いため）
```

## 🎯 N3での入力例

### 例1: 日本語版ポケモンカード
```
商品名: ポケモンカード リザードン VMAX PSA10
原産国: JP
素材: Card Stock, Plastic Sleeve
HTSコード: 9504.40.00.00
```

### 例2: 英語版ポケモンカード（米国製）
```
商品名: Pokemon Card Charizard VMAX PSA10 (English)
原産国: US
素材: Card Stock, Plastic Sleeve
HTSコード: 9504.40.00.00
```

### 例3: 英語版ポケモンカード（製造国不明）
```
商品名: Pokemon Card Charizard VMAX PSA10 (English)
原産国: CN  ← 不明な場合は中国と仮定
素材: Card Stock, Plastic Sleeve
HTSコード: 9504.40.00.00
```

## 📊 関税率の違い

| 原産国 | HTSコード | 関税率 | 備考 |
|--------|----------|--------|------|
| JP（日本） | 9504.40.00.00 | Free | 日米貿易協定 |
| US（米国） | 9504.40.00.00 | Free | - |
| CN（中国） | 9504.40.00.00 | Free | - |
| KR（韓国） | 9504.40.00.00 | Free | 米韓FTA |

**Playing cards (9504.40.00.00) はほぼ全ての国から無税です。**

---

## 💡 実務での対応

### 原則
1. **確認できる場合**: 実際の製造国を入力
2. **不明な場合**: `CN`（中国）と入力
3. **疑義がある場合**: 商品説明に「製造国確認中」と記載

### リスク回避
- 関税率が`Free`の場合は製造国の誤りによる追加課税リスクは低い
- ただし、税関申告書には正確な情報が必要
- **PSAグレード品は米国（US）で評価されるが、カード自体の製造国とは別**
