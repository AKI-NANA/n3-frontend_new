🚀 サーバー起動手順

⚠️ 重要: このシステムはPHPサーバーが必要です。以下の手順でサーバーを起動してください。

📂 ステップ1: ディレクトリに移動

cd /Users/aritahiroaki/NAGANO-3/N3-Development/modules/yahoo_auction_complete/new_structure/05_rieki/

🔧 ステップ2: PHPサーバー起動

以下のいずれかのコマンドを実行:

# 方法1: スクリプト使用
chmod +x start_server.sh
./start_server.sh
# 方法2: 直接起動
php -S localhost:8080
# 方法3: 代替ポート
php -S localhost:8080

🌐 ステップ3: ブラウザでアクセス

サーバー起動後、以下のURLにアクセス:

http://localhost:8080/advanced_tariff_calculator.html

代替ポートの場合:

http://localhost:8080/advanced_tariff_calculator.html

🔍 トラブルシューティング

PHPがインストールされていない場合:

# macOS (Homebrew)
brew install php
# Ubuntu/Debian
sudo apt update
sudo apt install php

ポートが使用中の場合:

# 使用中のポートを確認
lsof -i :8080
# 別のポートで起動
php -S localhost:8082
