Shopify商品説明エディタ
プレビュー

商品説明を入力するとプレビューが表示されます

Shopify商品説明のベストプラクティス
  • HTMLサポート: すべての標準HTMLタグが使用可能
  • Liquid記法: {{ product.title }} などの変数が使用可能
  • レスポンシブ: モバイル表示を考慮したデザイン推奨
  • 画像: CDN経由で高速読み込み（imgタグ使用可）
  • CSS: インラインスタイルまたは<style>タグ使用可
  • JavaScript: 制限あり（テーマファイルでの実装推奨）
よく使うLiquid変数
{{ product.title }} - 商品名
{{ product.price }} - 価格
{{ product.vendor }} - ベンダー名
{{ product.type }} - 商品タイプ
{{ product.images[0] }} - 1枚目画像
{{ product.tags }} - タグ一覧
