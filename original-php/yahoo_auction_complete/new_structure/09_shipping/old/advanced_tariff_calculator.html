計算式・ロジック表示
eBay USA DDP計算式
1. 総コスト計算:
総コスト円 = 仕入れ価格 + 外注工賃費 + 梱包費 + 国内送料
2. 為替変動考慮:
安全為替レート = 基本為替レート × (1 + 為替変動マージン%)
3. 関税計算 (DDP):
関税額USD = (販売価格USD + 送料USD) × 関税率%
4. eBay手数料:
Final Value Fee = 収入総額 × 12.9% (標準)
PayPal手数料 = 収入総額 × 3.49% + $0.49
5. 最終利益:
利益円 = (収入総額USD - 関税額USD - eBay手数料USD) × 安全為替レート - 総コスト円
eBay USA DDU計算式
DDU特記事項:
関税は買主負担のため、売主の利益計算には含めない
ただし、返品リスクや価格競争力への影響を考慮する必要あり
最終利益 (DDU):
利益円 = (収入総額USD - eBay手数料USD) × 安全為替レート - 総コスト円
Shopee 7カ国計算式
1. 総コスト計算:
総コスト円 = 仕入れ価格 + 外注工賃費 + 梱包費 + 国際送料
2. 関税・税計算:
課税対象額 = 販売価格 + 送料 - 免税額
関税額 = max(0, 課税対象額 × 関税率%)
GST/VAT額 = (課税対象額 + 関税額) × GST/VAT率%
3. Shopee手数料:
販売手数料 = 収入総額 × 販売手数料率% (国別)
決済手数料 = 収入総額 × 2%
4. 最終利益:
利益円 = (収入総額 - 関税額 - GST/VAT額 - Shopee手数料) × 為替レート - 総コスト円
