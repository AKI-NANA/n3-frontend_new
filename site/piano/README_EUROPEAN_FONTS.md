## 🇪🇺 厳選フォント一覧

### 1. **Bodoni Moda** 🇮🇹
- **起源**: イタリア (1798年)
- **デザイナー**: Giambattista Bodoni
- **特徴**: ハイコントラスト、優雅、格調高い
- **おすすめ**: ⭐⭐⭐⭐⭐ クラシック音楽に最適!

### 2. **Italiana** 🇮🇹
- **起源**: イタリア (現代)
- **特徴**: 細身で繊細、ピアノの音色のような美しさ
- **おすすめ**: ⭐⭐⭐⭐⭐ 洗練されたモダン

### 3. **Sorts Mill Goudy** 🇬🇧
- **起源**: イギリス (1915年)
- **デザイナー**: Frederic Goudy
- **特徴**: 品格、温かみ、ヴィクトリア時代の優雅さ
- **おすすめ**: ⭐⭐⭐⭐ 落ち着いた英国スタイル

### 4. **Libre Caslon Text** 🇫🇷
- **起源**: フランス (1734年)
- **デザイナー**: William Caslon
- **特徴**: バロック時代の華やかさと優雅さ
- **おすすめ**: ⭐⭐⭐⭐ フランス宮廷の気品

### 5. **Cardo** 🇮🇹
- **起源**: イタリア・ルネサンス
- **特徴**: 知的、古典美、ヒューマニスト書体
- **おすすめ**: ⭐⭐⭐⭐ 学術的で美しい

### 6. **Crimson Pro** 🇪🇺
- **起源**: ヨーロッパ (現代)
- **特徴**: モダンとクラシックの融合、読みやすい
- **おすすめ**: ⭐⭐⭐⭐ バランスが良い

### 7. **Eczar** 🇮🇳
- **起源**: インド・ラテン融合
- **特徴**: 力強さとエレガンスの調和
- **おすすめ**: ⭐⭐⭐ ユニーク

### 8. **Alegreya** 🇪🇸
- **起源**: スペイン
- **特徴**: カリグラフィック、温かみ
- **おすすめ**: ⭐⭐⭐⭐ 親しみやすい

### 9. **Yeseva One** 🇷🇺
- **起源**: ロシア帝国
- **特徴**: ドラマティック、華麗
- **おすすめ**: ⭐⭐⭐ 華やか

### 10. **Marcellus** 🏛️
- **起源**: 古代ローマ
- **特徴**: 碑文書体、永遠の美しさと威厳
- **おすすめ**: ⭐⭐⭐⭐ 格式高い

### 11. **Cormorant Garamond** 🇫🇷
- **起源**: フランス (1530年)
- **デザイナー**: Claude Garamond
- **特徴**: 細身で優雅、洗練の極み
- **おすすめ**: ⭐⭐⭐⭐⭐ フレンチエレガンス!

### 12. **Cinzel** 🏛️ (現在使用中)
- **起源**: ローマ帝国碑文
- **特徴**: 格式高い、高級感、エレガント
- **おすすめ**: ⭐⭐⭐⭐⭐ 現在の完璧な選択!

## 💡 フォント変更方法

### 例: Bodoni Modaに変更
```css
/* /css/style-fixes.css */

@import url('https://fonts.googleapis.com/css2?family=Bodoni+Moda:wght@400;600;700&display=swap');

.hero-title {
  font-family: 'Bodoni Moda', serif !important;
  letter-spacing: 0.08em;
  font-weight: 600;
}

.logo {
  font-family: 'Bodoni Moda', serif !important;
  letter-spacing: 0.08em;
}

.phone-link {
  font-family: 'Bodoni Moda', serif;
}
```

### 例: Cormorant Garamondに変更
```css
@import url('https://fonts.googleapis.com/css2?family=Cormorant+Garamond:wght@300;400;600&display=swap');

.hero-title {
  font-family: 'Cormorant Garamond', serif !important;
  letter-spacing: 0.05em;
  font-weight: 600;
}

.logo {
  font-family: 'Cormorant Garamond', serif !important;
}

.phone-link {
  font-family: 'Cormorant Garamond', serif;
}
```

### 例: Italianaに変更
```css
@import url('https://fonts.googleapis.com/css2?family=Italiana&display=swap');

.hero-title {
  font-family: 'Italiana', serif !important;
  letter-spacing: 0.1em;
}

.logo {
  font-family: 'Italiana', serif !important;
}

.phone-link {
  font-family: 'Italiana', serif;
}
```
