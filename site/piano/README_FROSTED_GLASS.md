### 2. すりガラス効果

```css
backdrop-filter: blur(20px) saturate(180%);
-webkit-backdrop-filter: blur(20px) saturate(180%);
```

**効果**:
- `blur(20px)` - 背景を強くぼかす
- `saturate(180%)` - 色を1.8倍鮮やかに
- `-webkit-` - Safari対応

**視覚効果**:
- すりガラスのような質感
- 背景がぼんやり透ける
- 高級感のある仕上がり

---

### 3. 文字色

```css
color: white !important;
text-shadow: 0 2px 8px rgba(0,0,0,0.5);
```

**理由**:
- 背景が薄いので白文字が最適
- 濃い影で視認性確保
- 初期状態と統一感

## 🌟 すりガラスの秘密

### backdrop-filter の仕組み:
```
通常の blur:
背景 → ヘッダー背景色 → ぼかし → 表示

backdrop-filter:
背景 → ぼかし → ヘッダー背景色 → 表示
        ↑
    すりガラス効果!
```

**違い**:
- 通常: ヘッダー自体がぼやける
- backdrop-filter: **背景がぼやける** (すりガラス)
