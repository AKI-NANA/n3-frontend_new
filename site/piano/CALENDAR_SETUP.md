# Googleカレンダー設定ガイド

## Googleカレンダーの公開とWebサイトへの埋め込み手順

### ステップ1: Googleカレンダーの作成

1. [Google Calendar](https://calendar.google.com/)にアクセス
2. 左側のサイドバーで「マイカレンダー」の横にある「+」をクリック
3. 「新しいカレンダーを作成」を選択
4. カレンダー情報を入力:
   - **名前**: 山内ピアノ教室レッスンスケジュール
   - **説明**: レッスン可能日時を表示するカレンダー
   - **タイムゾーン**: (GMT+09:00) 東京
5. 「カレンダーを作成」をクリック

### ステップ2: カレンダーを公開設定

1. 左側の「マイカレンダー」で作成したカレンダーにマウスを合わせる
2. 「⋮」(縦3点)アイコンをクリック
3. 「設定と共有」を選択
4. 「アクセス権限」セクションで以下を設定:
   - ☑ **「一般公開して誰でも利用できるようにする」にチェック**
   - 警告が表示されますが、「OK」をクリック
   - ⚠️ 重要: プライベートな情報は入れないようにしてください

### ステップ3: Calendar IDの取得

1. 同じ設定画面を下にスクロール
2. 「カレンダーの統合」セクションを見つける
3. **「カレンダーID」**をコピー
   - 例: `abc123def456@group.calendar.google.com`
4. このIDは後で使用するのでメモしておいてください

### ステップ4: 埋め込みコードのカスタマイズ

1. 設定画面をさらに下にスクロール
2. 「埋め込みコード」セクションで「カスタマイズ」をクリック
3. 以下の設定を推奨します:

#### 表示オプション
- ☐ **タイトルを表示**: OFF (Webサイト側でタイトルを表示)
- ☑ **ナビゲーションボタンを表示**: ON
- ☑ **日付を表示**: ON
- ☐ **印刷アイコンを表示**: OFF
- ☐ **タブを表示(週、月、予定リスト)**: OFF
- ☐ **カレンダーリストを表示**: OFF
- ☐ **タイムゾーンを表示**: OFF

#### 外観
- **幅**: 800 (ピクセル)
- **高さ**: 600 (ピクセル)
- **背景色**: #ffffff
- **枠線の幅**: 0

4. 「コードを更新」をクリック

### ステップ5: HTMLファイルへの適用

#### 方法A: 生成されたコードをそのまま使用

1. カスタマイズ画面で生成されたiframeコードをコピー
2. `yamauchi_piano_complete_v2.html`の1101行目付近を以下のように置き換え:

```html
<!-- 元のコード(削除) -->
<iframe 
  src="https://calendar.google.com/calendar/embed?height=600&wkst=1&ctz=Asia%2FTokyo&bgcolor=%23ffffff&src=YOUR_CALENDAR_ID&color=%23B39DDB&mode=MONTH&showTitle=0&showNav=1&showDate=1&showPrint=0&showTabs=0&showCalendars=0&showTz=0" 
  style="border:solid 1px #777" 
  frameborder="0" 
  scrolling="no">
</iframe>

<!-- 新しいコード(生成されたものをペースト) -->
<iframe src="生成されたURL" ...></iframe>
```

#### 方法B: Calendar IDのみを置き換え(推奨)

1. `yamauchi_piano_complete_v2.html`の1101行目付近
2. `YOUR_CALENDAR_ID`の部分だけを、コピーしたCalendar IDに置き換え

**置き換え前:**
```html
src="https://calendar.google.com/calendar/embed?height=600&wkst=1&ctz=Asia%2FTokyo&bgcolor=%23ffffff&src=YOUR_CALENDAR_ID&color=%23B39DDB&mode=MONTH&showTitle=0&showNav=1&showDate=1&showPrint=0&showTabs=0&showCalendars=0&showTz=0"
```

**置き換え後:**
```html
src="https://calendar.google.com/calendar/embed?height=600&wkst=1&ctz=Asia%2FTokyo&bgcolor=%23ffffff&src=abc123def456@group.calendar.google.com&color=%23B39DDB&mode=MONTH&showTitle=0&showNav=1&showDate=1&showPrint=0&showTabs=0&showCalendars=0&showTz=0"
```

### ステップ6: レッスンスケジュールの追加

Googleカレンダーに実際のレッスン可能日時を追加します:

1. Googleカレンダーのメイン画面に戻る
2. 作成したカレンダーが選択されていることを確認
3. 日付をクリックして予定を追加

#### 例: 定期レッスン枠の追加

**火曜日 14:00-15:00**
- タイトル: レッスン可能
- 日時: 毎週火曜日 14:00-15:00
- 繰り返し: 毎週火曜日
- カレンダー: 山内ピアノ教室レッスンスケジュール
- 色: 任意(例: 薄紫)
- 説明: (任意)体験レッスン・通常レッスン可能時間

**水曜日 16:00-18:00**
- タイトル: レッスン可能
- 日時: 毎週水曜日 16:00-18:00
- 繰り返し: 毎週水曜日

**土曜日 10:00-12:00**
- タイトル: レッスン可能
- 日時: 毎週土曜日 10:00-12:00
- 繰り返し: 毎週土曜日

#### 休講日の追加

特定の日に休講がある場合:

- タイトル: 休講
- 日時: 該当日時
- 終日: (チェック)
- カレンダー: 山内ピアノ教室レッスンスケジュール
- 色: 赤系
- 説明: お盆休み/年末年始休講 など

### ステップ7: テストと確認

1. HTMLファイルをブラウザで開く
2. 「レッスン日程」セクションまでスクロール
3. カレンダーが正しく表示されることを確認
4. カレンダーの操作(月の移動、日付クリック)が機能することを確認

### トラブルシューティング

#### カレンダーが表示されない

**原因1: Calendar IDが正しくない**
- Googleカレンダーの設定画面で再度Calendar IDを確認
- コピー&ペーストミスがないか確認(スペースや改行が入っていないか)

**原因2: カレンダーが非公開**
- 「アクセス権限」で「一般公開して誰でも利用できるようにする」がチェックされているか確認

**原因3: ブラウザのセキュリティ設定**
- 別のブラウザで試す
- シークレットモードで試す

#### カレンダーは表示されるがスタイルが崩れている

- iframeのwidth/heightを調整
- CSSのカレンダーコンテナのサイズを調整

#### カレンダーが古い情報を表示している

- ブラウザのキャッシュをクリア(Ctrl+Shift+Delete)
- スーパーリロード(Ctrl+F5 または Cmd+Shift+R)

### セキュリティとプライバシー

#### 公開カレンダーの注意点

✅ **公開しても良い情報:**
- レッスン可能時間帯
- 一般的な休講日(お盆、年末年始など)
- 発表会などの公開イベント

❌ **公開してはいけない情報:**
- 生徒さんの個人名
- 生徒さんの具体的なレッスン時間
- 個人的な予定やプライベートな情報
- 自宅の詳細な住所

#### 推奨設定

レッスンスケジュールには「レッスン可能」「休講」など一般的な情報のみを記載し、具体的な生徒さんの予約は別の非公開カレンダーで管理することをお勧めします。

### カレンダーURLパラメータの説明

埋め込みURLに含まれるパラメータの意味:

- `height=600`: カレンダーの高さ(ピクセル)
- `wkst=1`: 週の開始日(1=月曜日、2=日曜日)
- `ctz=Asia%2FTokyo`: タイムゾーン(東京)
- `bgcolor=%23ffffff`: 背景色(白)
- `src=`: カレンダーID
- `color=`: カレンダーの色
- `mode=MONTH`: 表示モード(MONTH/WEEK/AGENDA)
- `showTitle=0`: タイトル非表示(0)
- `showNav=1`: ナビゲーション表示(1)
- `showDate=1`: 日付表示(1)
- `showPrint=0`: 印刷アイコン非表示(0)
- `showTabs=0`: タブ非表示(0)
- `showCalendars=0`: カレンダーリスト非表示(0)
- `showTz=0`: タイムゾーン非表示(0)

### よくある質問

**Q: スマートフォンでも見やすく表示されますか?**
A: はい。レスポンシブ対応しているため、画面サイズに応じて自動調整されます。

**Q: カレンダーの色を変更できますか?**
A: はい。URLの`color=`パラメータを変更するか、Googleカレンダー側の設定で色を変更できます。

**Q: 複数のカレンダーを同時に表示できますか?**
A: はい。複数のカレンダーIDを`&src=`パラメータで追加することで可能です。

**Q: カレンダーの予定を直接Webサイトから追加できますか?**
A: いいえ。埋め込みカレンダーは閲覧のみです。予定の追加はGoogleカレンダー側で行います。

**Q: 過去の予定を非表示にできますか?**
A: URLパラメータでは制御できませんが、Googleカレンダー側で過去の予定を削除するか、新しいカレンダーを作成して今後の予定のみを追加することができます。
