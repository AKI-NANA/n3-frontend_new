## 💡 技術的なポイント

### 質感の実装方法
```css
/* 各セクションに::before疑似要素で質感を追加 */
.section::before {
  content: '';
  position: absolute;
  top: 0;
  left: 0;
  width: 100%;
  height: 100%;
  background: [質感パターン];
  pointer-events: none;
  z-index: 0;
}

/* コンテンツを前面に */
.section-container {
  position: relative;
  z-index: 1;
}
```

### 利点
- ✅ HTMLを変更せずにCSSだけで実装
- ✅ 軽量 (画像不要)
- ✅ レスポンシブ対応
- ✅ 読み込み速度に影響なし
